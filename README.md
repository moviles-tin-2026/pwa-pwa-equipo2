Proyecto Final: PyME-Sync: Sistema Móvil de Gestión de Inventario y Ventas.
PyME-Sync es el proyecto integrador final del curso y tiene como propósito que el estudiante diseñe, desarrolle
y despliegue una solución móvil multiplataforma orientada a la operación de un micronegocio, integrando
autenticación, inventario, ventas y visualización administrativa en la nube. El proyecto articula progresivamente
los aprendizajes construidos en las cuatro prácticas integradoras del cuatrimestre: diseño de interfaces,
arquitectura con Flutter y Dart, integración con Firebase y despliegue web profesional.
Justificación de negocio
En un micronegocio, la falta de control sobre inventario, ventas y disponibilidad de productos genera errores
operativos, pérdidas y decisiones basadas en suposiciones. PyME-Sync responde a esta problemática mediante
una aplicación que centraliza la información del negocio en tiempo real, permitiendo registrar productos,
controlar existencias, capturar ventas y consultar datos operativos desde entornos móviles y web. Desde la
perspectiva formativa, el proyecto no se limita al desarrollo de una app, sino que exige al estudiante modelar
procesos reales de negocio y convertirlos en una solución digital usable, conectada y desplegable. Esto fortalece
un perfil profesional orientado a resolver necesidades empresariales mediante tecnologías móviles, servicios
cloud y prácticas actuales de ingeniería de software.
Alcance del MVP
El producto mínimo viable deberá integrar cuatro módulos obligatorios alineados con la secuencia didáctica del
curso.
* Seguridad: autenticación con Firebase Auth mediante registro, inicio de sesión, cierre de sesión y
persistencia de sesión.
* Inventario reactivo: CRUD completo de productos en Cloud Firestore con lectura en tiempo real y carga
de imágenes en Firebase Storage.
* Transacciones: registro de ventas y reducción automática del stock, evitando inconsistencias
operativas.
* Dashboard y despliegue: compilación para entorno web con flutter build web y publicación en Firebase
Hosting con URL funcional.
